# check-site-vk-teams-bot
### Setup

- Что бы запустить проект используйте **Python 3.8**
- Получите api ключ можно с этого бота [Метаботу](https://teams.vk.com/profile/70001)
- Переименуйте файл .env-TEMP в .env подставте значение ключа вместо иксов
- Запустите код проверте что все корректно обрабатывается
